## Development

### Requirements

* jdk >= 11
* maven >= 3.6.3

`rtomde` uses and recommends the `IntelliJ IDEA` to develop the `rtomde` project. Minimal requirements for an IDE are java and maven supports.

### Build

```shell
./mvnw clean install
```
